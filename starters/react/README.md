# Typewriting Class React Starter

Minimal React + Typewriting Class project template.

## Quick start

```bash
# Copy this starter
bunx degit typewritingclass/typewritingclass/starters/react my-app
cd my-app

# Install dependencies
bun install

# Start dev server
bun dev
```

## What's included

- React 19 + Vite 6
- typewritingclass
- typewritingclass-compiler (Vite plugin)
- TypeScript 5.7
- Example `App.tsx` with a hover card demo

## Scripts

| Command | Description |
|---|---|
| `bun dev` | Start development server |
| `bun run build` | Production build |
| `bun run preview` | Preview production build |
