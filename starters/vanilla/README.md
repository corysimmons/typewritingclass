# typewritingclass Vanilla Starter

Minimal vanilla TypeScript + typewritingclass project template. No framework â€” just the DOM and typewritingclass.

## Quick start

```bash
# Copy this starter
bunx degit typewritingclass/typewritingclass/starters/vanilla my-app
cd my-app

# Install dependencies
bun install

# Start dev server
bun dev
```

## What's included

- Vanilla TypeScript + Vite 6
- typewritingclass
- typewritingclass-compiler (Vite plugin, strict mode off)
- TypeScript 5.7
- Example `main.ts` with a hover card demo
