# typewritingclass React Starter

Minimal React + typewritingclass project template.

## Quick start

```bash
# Copy this starter
npx degit typewritingclass/typewritingclass/starters/react my-app
cd my-app

# Install dependencies
pnpm install

# Start dev server
pnpm dev
```

## What's included

- React 19 + Vite 6
- typewritingclass + typewritingclass-react
- typewritingclass-compiler (Vite plugin, strict mode off)
- TypeScript 5.7
- Example `App.tsx` with a hover card demo

## Scripts

| Command | Description |
|---|---|
| `pnpm dev` | Start development server |
| `pnpm build` | Production build |
| `pnpm preview` | Preview production build |
