## 1. Install

<Tabs syncKey="pkg">
  <TabItem label="pnpm">
    ```bash
    pnpm add typewritingclass
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash
    bun add typewritingclass
    ```
  </TabItem>
  <TabItem label="npm">
    ```bash
    npm install typewritingclass
    ```
  </TabItem>
</Tabs>

## 2. Import

```ts
// src/main.ts (or main.tsx, index.tsx, etc.)
import 'typewritingclass/preflight.css' // optional CSS reset
import 'typewritingclass/inject'
```

## 3. Style

```tsx
import { tw } from 'typewritingclass'

function App() {
  return <div className={`${tw.p(6).bg.blue500.textColor.white.rounded.lg}`}>It works!</div>
}
```

Start your dev server. You should see a blue rounded box with white text.

No build plugin, no config file, no generated CSS. Just install, import, and go.

## Next.js

Next.js requires build-time CSS extraction for SSR. See the [Next.js integration guide](/integrations/nextjs/).

## Static extraction (optional)

For production optimization, the compiler can extract CSS at build time for zero-runtime overhead. See the [Vite plugin](/compiler/vite/), [esbuild plugin](/compiler/esbuild/), or [Babel plugin](/compiler/babel/) docs.
